# Voltex

Voltex is a tech mod for Minecraft 1.20.1 (Forge). It adds machines, FE energy generation and storage, cables and fluid handling, item and fluid logistics, a multiblock miner, a multiblock nuclear reactor, a full line of solar panels, two armor sets and a number of optional machines.

Downloads, screenshots and the full description are on CurseForge:
https://www.curseforge.com/minecraft/mc-mods/voltex

## Dependencies

- Minecraft 1.20.1, Forge 47.x
- Patchouli (required, in-game guide book)
- JEI (optional, recipe viewer)
- KubeJS (optional, recipe integration for pack makers)

## Building

Standard ForgeGradle. From the project root:

    ./gradlew build

The built jar is placed in `build/libs/`.
